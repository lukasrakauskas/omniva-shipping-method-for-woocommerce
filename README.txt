Adds Omniva self-service parcel terminals for WooCommerce.

== Description ==

This plugins adds Omniva (omniva.lt) terminal shipping method for WooCommerce.

It supports Omniva shipping to self-service parcel terminals which are based in Lithuania, Latvia and Estonia

Customer can pick nearest parcel terminal in checkout. Terminals' list is auto-updated so it's always up-to-date.

== Installation ==

1. Upload `omniva-shipping-method-for-woocommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce -> Settings
4. In the top of the page choose 'Shipping'
5. Add shipping zone and add shipping method in it
